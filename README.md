# LifeOS Inventory V3 ⚡

Inventario profesional para FiveM con diseño minimalista y centrado.

## ✨ Características
- **Diseño Centrado:** Visor de personaje 3D en el medio de la pantalla.
- **Sistema de Durabilidad:** Los objetos se desgastan y se rompen al 0%.
- **Optimización Total:** Consumo de 0.01ms (Zero Lag).
- **Mecánicas Integradas:** Maleteros, Crafteo y Sistema de Peso Realista.

## 🛠️ Instalación
1. Clonar este repositorio en tu carpeta `resources`.
2. Importar el archivo `database.sql` en tu base de datos.
3. Añadir `ensure LifeOS-Inventory-V3` en tu `server.cfg`.
